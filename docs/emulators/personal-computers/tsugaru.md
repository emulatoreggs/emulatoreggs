tsugaru is available as a binary file on Linux. This page will cover how to utilize and install the binary file. 

To download tsugaru:

1. Open the tsugaru [repository](github.com/schellingb/dosbox-pure-unleashed/releases/latest) on a browser of your choice
3. Click the `Assets` button below the most recent release
3. Download the latest `ubuntu_binary_latest.zip` file
4. Move the newly downloaded `zip` file to the `$HOME/Applications` folder
    * {{ applications }}
5. Extract the file
6. Rename the newly extracted `ubuntu_binary_latest` folder to `tsugaru`
7. In the newly named `tsugaru` folder, locate the `Tsugaru_GUI` and `Tsugaru_CUI` files, {{ permissions }}
8. tsugaru will now be downloaded
