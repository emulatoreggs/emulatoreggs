## Getting Started with Neko Project II kai

### How to Install Neko Project II kai
[Back to the Top](#neko-project-ii-kai-table-of-contents)

Neko Project II kai is available as an **unofficial** AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download Neko Project II kai:

1. Open the Neko Project II kai **unofficial** GitHub releases page, [https://github.com/pkgforge-dev/Neko-Project-II-Kai-AppImage/releases](https://github.com/pkgforge-dev/Neko-Project-II-Kai-AppImage/releases) :material-information-outline:{ title="{{ steamdeckdesktopmode }}" }
2. Download the `Neko_Project_II_kai-anylinux-x86_64.AppImage` file
3. Move the downloaded AppImage to `$HOME/Applications`.
4. Right click the AppImage, {{ permissions }}
5. Double click the AppImage to run Neko Project II kai. 
6. (Optional) For easier maintenance, rename the AppImage to `Neko_Project_II_kai.AppImage`
