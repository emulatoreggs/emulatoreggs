FS-UAE is available as a binary file on Linux. This page will cover how to utilize and install the binary file. 

To download FS-UAE:

1. Open the FS-UAE [website](https://fs-uae.net/download/linux/) on a browser of your choice
2. Download both files in the `x86-64` box
3. Move the two newly downloaded `tar.gz` files to the `$HOME/Applications` folder
    * {{ applications }}
4. Extract the two files
5. In `$HOME/Applications`, create a `fs-uae` folder and move the two newly extracted folders to this newly created folder
5. Locate the newly extracted `fs-uae-launcher` file in the `FS-UAE-Launcher` folder, {{ permissions }}
6. FS-UAE will now be downloaded
