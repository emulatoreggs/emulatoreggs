### How to Install DOSBox-X

DOSBox-X is available as a Flatpak on Linux. This page will cover how to utilize and install the Flatpak. 

To download DOSBox-X:

1. Open your distro's software manager.
    * {{ discover }}
2. Search for "DOSBox-X" and click "Install" on the top right of the software page For DOSBox-X
3. Open Konsole or a terminal of your choice
4. Type the following command and press enter after typing the command:
    * `flatpak override com.dosbox_x.DOSBox-X --filesystem=host --user`
    * {{ flatseal }}
5. DOSBOX-X will now be installed and properly configured
