DREAMM is available as a binary file on Linux. This page will cover how to utilize and install the binary file. 

To download DREAMM:

1. Open the DREAMM [website](https://dreamm.aarongiles.com/) on a browser of your choice
2. Click `Download for x64 Linux`
3. Move the newly downloaded `tgz` file to the `$HOME/Applications` folder
    * {{ applications }}
4. Extract the file
4. Rename the extracted folder to `dreamm`
5. In the newly named `dreamm` folder, {{ permissions }}
6. DREAMM will now be downloaded
