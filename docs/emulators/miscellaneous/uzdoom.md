### How to Install UZDoom

UZDoom is available as a Flatpak on Linux. This page will cover how to utilize and install the Flatpak. 

To download UZDoom:

1. Open your distro's software manager.
    * {{ discover }}
2. Search for "UZDoom" and click "Install" on the top right of the software page For UZDoom
3. Open Konsole or a terminal of your choice
4. Type the following two lines, one at a time, and press enter after each line:
    * `flatpak override org.zdoom.UZDoom --filesystem=host --user`
    * {{ flatseal }}
