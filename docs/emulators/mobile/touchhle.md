## Getting Started with touchHLE

### How to Install touchHLE
[Back to the Top](#touchhle-table-of-contents)

touchHLE is available as an **unofficial** AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download touchHLE:

1. Open the touchHLE **unofficial** GitHub releases page, [https://github.com/pkgforge-dev/TouchHLE-AppImage/releases](https://github.com/pkgforge-dev/TouchHLE-AppImage/releases) :material-information-outline:{ title="{{ steamdeckdesktopmode }}" }
2. Download the `touchHLE-v#.#.#-anylinux-x86_64.AppImage` file
   * The #.#.# may vary depending on when you download TouchHLE 
3. Move the downloaded AppImage to `$HOME/Applications`.
4. Right click the AppImage, {{ permissions }}
5. Double click the AppImage to run touchHLE. 
6. (Optional) For easier maintenance, rename the AppImage to `touchHLE.AppImage`
