EKA2L1 is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download EKA2L1:

1. Open the EKA2L1 [repository](https://github.com/EKA2L1/EKA2L1/releases/tag/continous) on a browser of your choice
2. Download the latest `EKA2L1-Linux-x86_64.AppImage` file
3. Move the newly downloaded AppImage to the `$HOME/Applications` folder
    * {{ applications }}
4. {{ permissions }}
5. EKA2L1 will now be downloaded
