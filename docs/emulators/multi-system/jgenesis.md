# jgenesis is a multi-console emulator supporting a number of 8-bit and 16-bit gaming consoles.

??? info "Systems Supported"

    * Nintendo NES
    * Nintendo SNES
    * Nintendo Game Boy
    * Nintendo Game Boy Color
    * Nintendo Game Boy Advance
    * Sega CD, Sega 32X
    * Sega SG-1000
    * Sega Genesis

jgenesis is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download jgenesis:

1. Open the jgenesis GitHub releases page, [https://github.com/jsgroth/jgenesis/releases](https://github.com/jsgroth/jgenesis/releases) on a browser of your choice
2. Download the latest `jgenesis-gui_#.##.#_x86_64.AppImage` file
    * Make sure you download the `gui` version 
    * The `#.##.#` refer to the version number, this may vary depending on when you download jgenesis
3. Move the newly downloaded AppImage to the `$HOME/Applications` folder
    * {{ applications }}
4. {{ permissions }}
5. jgenesis will now be downloaded
6. (Optional) For easier maintenance, rename the AppImage to `jgenesis.AppImage`
