Super ZSNES is available as a binary file on Linux. This page will cover how to utilize and install the binary file. 

To download Super ZSNES:

1. Open the Super ZSNES website, [https://www.zsnes.com/#downloads](https://www.zsnes.com/#downloads) on a browser of your choice
2. Click the `Download` button under `Linux`
3. Move the newly downloaded `tar.xz` file to the `$HOME/Applications` folder
    * {{ applications }}
4. Extract the `tar.xz` file
5. Rename the newly extracted folder to `SUPERZSNES`
6. In the newly named `SUPERZSNES` folder, locate the `SUPERZSNES` file, {{ permissions }}
7. Super ZSNES will now be downloaded
