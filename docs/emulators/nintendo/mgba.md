mGBA is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download mGBA:

1. Open the mGBA [website](https://mgba.io/downloads.html) on a browser of your choice
2. Download the `Linux (AppImage, 64-bit)` file
3. Move the newly downloaded AppImage to the `$HOME/Applications` folder
    * {{ applications }}
4. Rename the AppImage to `mGBA-appimage-x64.appimage`
5. {{ permissions }}
6. Ryujinx will now be downloaded
