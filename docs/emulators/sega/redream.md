redream is available as a binary file on Linux. This page will cover how to utilize and install the binary file. 

To download redream:

1. Open the redream [website](https://redream.io/download) on a browser of your choice
2. Click the `Assets` button under the most recent release
3. Click the Linux icon under the most recent `DEVELOPMENT RELEASES`
4. Move the newly downloaded `tar.gz` file to the `$HOME/Applications` folder
    * {{ applications }}
5. Extract the `tar.gz` file
6. Locate the newly extracted `redream` file, {{ permissions }}
6. redream will now be downloaded
