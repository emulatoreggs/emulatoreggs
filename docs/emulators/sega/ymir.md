ymir is available as a binary file on Linux. This page will cover how to utilize and install the binary file. 

To download ymir:

1. Open the ymir GitHub repository, [https://github.com/StrikerX3/Ymir/releases](https://github.com/StrikerX3/Ymir/releases) on a browser of your choice
2. Click the `Assets` button under the most recent release
3. Click `ymir-linux-x86_64-AVX2-v#.#.#.tar.xz` 
    * The `#.#.#` refer to the version number, this may vary depending on when you download ymir
4. Move the newly downloaded `tar.xz` file to the `$HOME/Applications` folder
    * {{ applications }}
5. Extract the `tar.xz` file
6. Rename the newly extracted folder to `ymir`
7. In the newly named `ymir` folder, locate the `ymir-sdl3` file, {{ permissions }}
8. ymir will now be downloaded
