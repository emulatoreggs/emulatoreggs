Melon Mix is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download Melon Mix:

1. Open the Melon Mix [repository](https://github.com/vitor251093/KHMelonMix/releases) on a browser of your choice
2. Download the latest `MelonMix-appimage-x86_64.zip` file
3. Move the newly downloaded `zip` file to the `$HOME/Applications` folder
    * {{ applications }}
4. Extract the `MelonMix-appimage-x86_64.zip` file
5. In the `MelonMix-appimage-x86_64` folder, locate the `MelonMix.AppImage` file, {{ permissions }}
6. Melon Mix will now be downloaded
