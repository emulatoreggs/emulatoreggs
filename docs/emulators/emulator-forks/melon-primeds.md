melonPrimeDS is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download melonPrimeDS:

1. Open the melonPrimeDS [repository](https://github.com/makinori/melonPrimeDS/releases) on a browser of your choice
2. Click the `Assets` button below the most recent release
2. Download the latest `melonPrimeDS-appimage-x86_64.zip` file
3. Move the newly downloaded `zip` file to the `$HOME/Applications` folder
    * {{ applications }}
4. Extract the `melonPrimeDS-appimage-x86_64.zip` file
5. {{ permissions }}
6. melonPrimeDS will now be downloaded
