## Getting Started with Supermodel

### How to Install Supermodel
[Back to the Top](#supermodel-table-of-contents)

Supermodel is available as an **unofficial** AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download Supermodel:

1. Open the Supermodel **unofficial** GitHub releases page, [https://github.com/pkgforge-dev/Supermodel-AppImage/releases](https://github.com/pkgforge-dev/Supermodel-AppImage/releases) :material-information-outline:{ title="{{ steamdeckdesktopmode }}" }
2. Download the `Supermodel-v#.#.#-anylinux-x86_64.AppImage` file
   * The #.#.# may vary depending on when you download Supermodel 
3. Move the downloaded AppImage to `$HOME/Applications`.
4. Right click the AppImage, {{ permissions }}
5. Double click the AppImage to run Supermodel. 
6. (Optional) For easier maintenance, rename the AppImage to `Supermodel.AppImage`
