Hypseus Singe is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download Hypseus Singe:

1. Open the Hypseus Singe releases page [https://github.com/DirtBagXon/hypseus-singe/releases](https://github.com/DirtBagXon/hypseus-singe/releases) on a browser of your choice
2. Download the latest `hypseus-singe_v#.##.#_AppImage.tar.gz` file
    * The `#.##.#` refer to the version number, this may vary depending on when you download Hypseus Singe
3. Move the newly downloaded `tar.gz` to the `$HOME/Applications` folder
    * {{ applications }}
4. Extract the `tar.gz` file. If it extracts to another `tar.gz` file, extract it once more
5. In the newly extracted `hypseus-singe` folder, {{ permissions }}
6. Hypseus Singe will now be downloaded
7. (Optional) For easier maintenance, rename the AppImage to `hypseus_singe.AppImage`
