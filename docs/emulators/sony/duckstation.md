DuckStation is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download DuckStation:

1. Open the DuckStation [repository](https://github.com/stenzek/duckstation/releases/tag/latest) on a browser of your choice
2. Download the latest `DuckStation-x64.AppImage` file
3. Move the newly downloaded AppImage to the `$HOME/Applications` folder
    * {{ applications }}
4. {{ permissions }}
5. DuckStation will now be downloaded
