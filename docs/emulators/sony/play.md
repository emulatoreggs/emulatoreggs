Play is available as an AppImage on Linux. This page will cover how to utilize and install the AppImage. 

To download Play:

1. Open the Play [website](https://purei.org/downloads.php) on a browser of your choice
2. Download the latest `Linux` file
3. Move the newly downloaded AppImage to the `$HOME/Applications` folder
    * {{ applications }}
4. Rename the AppImage to `Play!.AppImage`
5. {{ permissions }}
6. Play will now be downloaded
