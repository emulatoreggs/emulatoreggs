PPSSPP is available both as an AppImage and a Flatpak on Linux. This page will cover how to utilize and install the Flatpak. 

To download PPSSPP:

1. Open your distro's software manager.
    * {{ discover }}
2. Search for "PPSSPP" and click "Install" on the top right of the software page for PPSSPP
3. Open Konsole or a terminal of your choice
4. Type the following command and press enter after typing the command:
    * `flatpak override org.ppsspp.PPSSPP --filesystem=host --user`
    * {{ flatseal }}
5. PPSSPP will now be installed and properly configured
