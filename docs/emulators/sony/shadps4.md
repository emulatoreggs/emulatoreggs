### How to Install ShadPS4

ShadPS4 is available as a Flatpak on Linux. This page will cover how to utilize and install the Flatpak. 

To download ShadPS4:

1. Open your distro's software manager.
    * {{ discover }}
2. Search for "ShadPS4" and click "Install" on the top right of the software page For ShadPS4
3. Open Konsole or a terminal of your choice
4. Type the following command and press enter after typing the command:
    * `flatpak override net.shadps4.shadPS4 --filesystem=host --user`
    * {{ flatseal }}
5. ShadPS4 will now be installed and properly configured
